# Figra 产品需求定义

## 1. 产品定位

Figra 是本地优先的 macOS PDF 工具箱，面向科研阅读、论文整理、报告制作和日常 PDF 处理。当前版本采用双主线：

- 自动图表提取：使用内置 `pdffigures2` 从论文 PDF 中识别 Figure/Table。
- 页面级 PDF 提取/导出：用户手动预览、选择页面，并导出图片或新 PDF。

产品原则：

- 所有处理默认在本地完成，不上传 PDF、图片或元数据。
- 所有 PDF 操作默认生成新文件，不覆盖原 PDF。
- 正式 app 图表提取只使用内置 `pdffigures2`，不做 PDF 内嵌对象拆解。
- 正式 app 使用随包 JRE，不依赖系统 Java、Python 或 conda。
- 图片 DPI 提供固定档位 `150 / 300 / 600 / 900`，默认 `600`。

## 2. 信息架构

主窗口使用原生 macOS 结构：左侧 source-list 工具导航，右侧只展示当前工具工作区，避免卡片堆叠和页面挤压。

固定工具入口：

- 图表提取
- 页面提取
- 页面整理
- 合并 PDF
- 拆分 PDF
- 压缩 PDF
- 清除隐私
- PDF 加密
- 最近任务
- 输出设置

视觉要求：

- 默认浅色模式使用系统语义色和白底内容区，不使用大面积半透明灰遮罩。
- 每个工具都有明确空状态、运行中、失败和完成文案。
- 主工作区保持单任务焦点，缩略图、预览和操作区主次清楚。

## 3. 页面提取

用户目标：

- 选择或拖入单个 PDF。
- 看到全部页面缩略图和 PDF 大预览。
- 通过点击、Command 多选、Shift 连续多选或页码范围选择页面。
- 左侧缩略图选中时，右侧 PDF 预览同步跳转到该页。
- 左侧缩略图双击时，将该页按当前 DPI 复制为图片到剪贴板。
- 支持一键全选和清空选择。
- 将选中页面导出成图片或新 PDF。

功能需求：

- 支持页码范围输入，例如 `1-3, 8, 10-12`。
- 缩略图选择与页码范围文本同步。
- DPI 下拉固定为 `150 / 300 / 600 / 900`，默认 `600`。
- 导出动作包括 `保存为图片`、`保存为一个 PDF`、`每页保存为 PDF`、`按连续分组保存 PDF`。
- 页码非法时显示错误，不执行导出。

验收标准：

- 载入 PDF 后显示全部页面缩略图。
- 选择 `1,3,5` 后导出图片，输出分辨率随 DPI 档位变化。
- 导出一个 PDF 时页面顺序符合选中顺序。
- 每页导出 PDF 时输出数量等于选中页数。
- 原 PDF 不被修改。

## 4. 图表提取

用户目标：

- 选择或拖入一个论文 PDF。
- 使用内置 `pdffigures2` 自动提取 Figure/Table。
- 提取完成后看到摘要，不让主页面被结果缩略图挤满。
- 通过弹出图库集中查看和复用图片。

功能需求：

- 每次只处理一个 PDF。
- 使用 `.app/Contents/Resources/jre/bin/java` 执行 `.app/Contents/Resources/pdffigures2.jar`。
- 输出 PNG，DPI 使用当前下拉选择值。
- 输出文件夹带 PDF 文件名前缀，例如 `{PDF文件名}_pdffigures2/`。
- 单张 PNG 文件名不重复携带 PDF 文件名前缀。
- 输出目录已存在时自动追加时间戳，避免覆盖。
- 提取失败时保留原始日志。

验收标准：

- 内置 JRE 和 `pdffigures2.jar` 存在时能成功提取。
- 缺少 JRE 或 jar 时显示资源损坏提示。
- 未识别到 Figure/Table 时显示空结果摘要，不视为崩溃。
- `figures/` 下的 PNG 文件名不带 PDF 文件名前缀。

## 5. 图库面板

用户目标：

- 查看全部提取图片和表格。
- 点击缩略图后立即复制图片或表格到剪贴板。
- 对选中结果执行打开、定位、复制、移除和导出清单操作。

功能需求：

- 弹出式图库，不挤压主工作区。
- 图库提供关闭按钮。
- 左侧或顶部展示全部缩略图。
- 主体展示选中图片的大图预览。
- 点击缩略图默认复制图片到剪贴板。
- 提供图片/表格筛选。
- 提供 `打开图片`、`在 Finder 中显示`、`复制图片`、`从结果移除`、`导出清单 CSV`。

验收标准：

- 点击缩略图后可以粘贴到 Keynote、PowerPoint 或笔记应用。
- CSV 至少包含图片文件名和输出路径。
- 缩略图加载失败不影响其他图片展示。

## 6. 页面整理

用户目标：

- 从一个 PDF 中删除不需要的页面。
- 调整页面顺序后导出新 PDF。

功能需求：

- 支持选择或拖入单个 PDF。
- 显示页面列表和缩略图。
- 支持拖动重排和移除页面。
- 导出为新 PDF，不修改原 PDF。

验收标准：

- 导出页序符合当前列表顺序。
- 移除页面只影响导出副本。

## 7. 合并 PDF

用户目标：

- 添加多个 PDF。
- 调整顺序后导出一个新 PDF。

功能需求：

- 列表每项左侧显示拖动把手。
- 列表每项右侧提供 `上移`、`下移`、`移除`。
- 支持拖拽排序。
- 显示文件名、页数和大小。
- 导出为新 PDF，不修改原文件。

验收标准：

- 至少两个 PDF 可合并。
- 拖拽排序和右侧上移/下移都生效。
- 导出的 PDF 页序符合列表顺序。

## 8. 拆分 PDF

用户目标：

- 从长 PDF 中截取页面。
- 按单页或固定页数拆分。

功能需求：

- 复用页面提取的页面选择组件。
- 支持页码范围、缩略图多选、选中页导出为一个 PDF。
- 支持每页保存为单个 PDF。
- 支持每 N 页保存为一个 PDF。

验收标准：

- 页码范围非法时显示错误，不导出。
- 每页拆分输出数量正确。
- 每 N 页拆分时最后不足 N 页也导出。
- 原 PDF 不被修改。

## 9. 压缩 PDF

用户目标：

- 为分享或归档生成体积优化副本。

功能需求：

- 支持选择或拖入单个 PDF。
- 使用 PDFKit 重写结构生成优化副本。
- 明确提示当前不重采样页面内图片，不承诺扫描件强压缩。

验收标准：

- 导出新 PDF，不覆盖原文件。
- 显示导出前后的文件大小。

## 10. 清除隐私

用户目标：

- 分享 PDF 前清理作者、标题、创建工具、时间等 document info metadata。
- 知道工具能力边界，避免误以为页面正文也被扫描。

功能需求：

- 显示 metadata 字段预览。
- 支持选择或拖入单个 PDF。
- 一键导出清理副本。
- 导出后可在 Finder 中定位清理副本。
- 不覆盖原 PDF。
- 明确提示只清除 PDF document info metadata，不承诺删除页面内容里的隐私文字。

验收标准：

- 清理副本不包含常见 document info 字段。
- 原 PDF 保持不变。
- 没有 metadata 时显示明确空状态。

## 11. PDF 加密

用户目标：

- 为 PDF 设置打开密码。
- 对已知密码的 PDF 导出无密码副本。

功能需求：

- 支持选择或拖入单个 PDF。
- 支持输入打开密码和可选所有者密码。
- 支持导出加密副本。
- 支持输入密码后导出无密码副本。
- 密码只在本次操作中使用，不保存。

验收标准：

- 原 PDF 不被修改。
- 密码错误时显示明确失败状态。

## 12. 最近任务和输出设置

用户目标：

- 快速回到刚才导出的文件或文件夹。
- 减少重复选择输出目录和 DPI 的操作。

功能需求：

- 记录本次运行中的导出历史。
- 支持打开或定位历史输出。
- 支持默认输出目录、默认 DPI 和导出后自动定位输出。

验收标准：

- 完成导出后历史列表出现记录。
- 默认输出目录会作为保存面板的初始目录。

## 13. 明确不做

当前版本不做：

- 云端同步、账号系统或上传处理。
- OCR、公式识别、表格结构重建。
- 批量论文管理器。
- 图表提取的多 PDF 队列。
- 删除页面正文、图片、批注中可见的隐私内容。
- 在当前压缩工具中对扫描图片进行有损重采样。
